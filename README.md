# dotfiles
These are my dotfiles. There are many like it, but these are mine

First, clone this repo to `~/dotfiles`.

Next, install [GNU Stow](https://www.gnu.org/software/stow/).

On Mac:

```shell
$ brew install stow
```

On Ubuntu:

```shell
$ sudo apt install stow
```

Then run the install script:

```shell
$ ./install
```
